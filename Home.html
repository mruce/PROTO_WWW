What is it? 

PROTOMAP is an Auto-Calibration System for Projection Mapping
WELL - WHAT DOES IT MEAN EXACTLY?

First things first:  Projection Mapping is a technique used to turn objects, often irregularly shaped, into a display surface for video projection. It means it can transform a physical object into a projection canvas and put life into it. Creating Mapping installations is hard - not only it requires artistic skills, but also tools and technical knowledge. PROTOMAP aims to solve most painful technical difficulties in the process, lowering the threshold for creators. Specifically - using PROTOMAP, you don't need to worry about technical aspects and focus on creating beautiful content. 

HOW IS IT DIFFERENT FROM OTHER PROJECTION MAPPING SYSTEMS?

There's couple of things:

  • Automatic calibration - that's the biggest one. Typically, to create a Projection Mapping application one needs to put a significant effort in calibrating the setup. It's difficult, prone to errors - and has to be repeated each time any element would changed or moved. With PROTOMAP all the calibration is automated - no more painful, manual calibration.
  • Use your favorite 3D rendering engine - well, if it's Unity, as for today. Think about it this way - you can create a Projection Mapping project in Unity, the same way as you'd make a 3D game - and all the actual mapping hassle is outsourced to PROTOMAP. Almost no extra skills required - just connect projectors and cameras and press a button. New engines will be added in the future
  • It spins! - our calibration is so good - and doesn't use any dirty tricks, like masking or warping - that we can even project content on moving objects. It's something that is very hard to impossible using traditional techniques
BUT HOW DOES IT WORK?

From the creator's point of view - you need to add webcams to your setup, run our software - and wait for calibration. Right now PROTOMAP works with Unity - which means you can create a 3D scene based on a model you're displaying on, place projectors and cameras - and run PROTOMAP. That's when magic happens.More technically - calibration is a three-step process:

  • Structured Light is used to match points on different projectors and cameras
  • Cameras' and projectors' optical and geometrical parameters are obtained using Genetic algorithm
  • 3D model's scale, position and rotation parameters are obtained using Genetic algorithm

Then, calibration is applied to the scene - and the show can go on.

And what are use cases?

PROTOMAP is a generic system - meaning it can work with arbitrary hardware setup. It pretty much means it can be used in all the scenarios Projection Mapping installations are used today - and more. So i t can be used in:

  • Museums
  • Interactive installations
  • Large scale outdoor shows
  • Much more

But! 
Due to PROTOMP's unique feature - auto-calibration - it can be used on large scale in new contexts - like retail. Imagine simple, easy to use Mapping installations that can be placed in Points Of Sale - everywhere in the world, really.

Right - but why would I want to use it?

There are two main benefits of using PROTOMAP:

  • You don’t need to worry about technical aspects of Projection Mapping to achieve great precision. It means, of course, that creating Mapping installation is getting cheaper - as you don’t need highly qualified staff to make it work
  • Even slight rearrangement of projectors makes installation look really bad - and it happens rather often. Because Calibration is now automatic, it can be fixed remotely, very fast. That's a great thing on it’s own - but we believe that it’s even more important: it makes it possible to use Mapping in contexts that were not feasible previously - for example it can be used in retail, in multiple locations at a time - without an army of staff to manage it.
